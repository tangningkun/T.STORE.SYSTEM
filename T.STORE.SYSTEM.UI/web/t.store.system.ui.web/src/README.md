### 文件提示 shims-tsx.d.ts shims-vue.d.ts

```ts
shims-tsx.d.ts ，允许你以 .tsx 结尾的文件，在 Vue 项目中编写 jsx 代码
shims-vue.d.ts 主要用于 TypeScript 识别 .vue 文件， ts 默认并不支持导入 .vue 文件，这个文件告诉 ts 导入 .vue 文件都按 VueConstructor<Vue> 处理。
```
